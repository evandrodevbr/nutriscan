## Dicas de Refinamento

### Parâmetros úteis para iteração no Midjourney

| Parâmetro | Uso |
|---|---|
| `--stylize 600–950` | Quanto maior, mais "opiniosa" e artística a saída. Para logos, 700–900 costuma funcionar bem. |
| `--ar 1:1` | Ideal para app icons e símbolos isolados |
| `--ar 2:1` | Ideal para wordmarks horizontais |
| `--ar 3:2` | Ideal para apresentação de marca completa |
| `--v 6` | Versão mais recente — mantém melhor a intenção do prompt |
| `--no shadows, gradients, 3D` | Adicione ao final para reforçar o estilo flat vector |

### Cores para referenciar nos prompts
- **Verde profundo:** `deep forest green`, `editorial green`, `#3a6b4a`
- **Fundo neutro:** `warm off-white`, `parchment white`, `cream`, `#f9f6f0`
- **Evitar:** `lime green`, `bright green`, `neon` — fogem da personalidade da marca
